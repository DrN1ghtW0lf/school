# DNA Replication

A prior necessity to Mitosis.
Before dividing, the cell must make identical copies of its DNA, it does this during the S phase.

DNA replication is semi-conservative.
```
| |             |  ||    |  ||
| | duplication |  ||    |  ||
| |   ------>   |  ||  + |  ||
| |             |  ||    |  ||
| |             |  ||    |  ||

| = original
|| = copy
```
